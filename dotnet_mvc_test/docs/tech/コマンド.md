# 便利コマンド

### マイグレーションの作成
```shell
dotnet ef migrations add InitialCreate
```

### データベースの更新
```shell
dotnet ef database update
```

### テスト
.csproj か .sln ファイルがあるディレクトリで以下のコマンドを実行します。
```shell
dotnet test

# フィルタリング（特定のテストだけ実行）
dotnet test --filter "FullyQualifiedName~UserService"
dotnet test --filter "Category=Integration"
```
